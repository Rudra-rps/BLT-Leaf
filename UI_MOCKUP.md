# UI Mockup

This document shows what the application interface looks like.

## Color Palette

The interface uses a GitHub-inspired dark theme:

```
Background:     #0d1117 (Very dark gray)
Cards:          #161b22 (Dark gray)
Borders:        #30363d (Medium gray)
Primary Links:  #58a6ff (Blue)
Success:        #238636 (Green)
Error:          #da3633 (Red)
Warning:        #d29922 (Yellow)
Text:           #c9d1d9 (Light gray)
Muted:          #8b949e (Medium light gray)
```

## Responsive Behavior

### Desktop (> 768px)
- Sidebar: 280px fixed width
- Main content: Flexible, fills remaining space
- PR cards: Full width with grid layout

### Mobile (< 768px)
- Sidebar: Collapsed or below main content
- Main content: Full width
- PR cards: Stacked vertically

## Accessibility Features

- Semantic HTML structure
- Proper heading hierarchy
- Alt text for images
- Keyboard navigation support
- ARIA labels where appropriate
- High contrast colors
- Focus indicators

## Animations

- Smooth transitions (0.2s) on:
  - Hover states
  - Badge appearances
  - Card borders
- No animations on data load (immediate update)
- Fade-out for error messages
